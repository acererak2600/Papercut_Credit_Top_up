# Papercut_Credit_Top_up
A simple Python script to top up Papercut user credit, utilizing the `xmlrpc.client` Python library to make API calls to Papercut.

## Setup

### Variables to change

| Variable | Description |
|---|---|
| `server_url` | Your Papercut server URL |
| `amount` | Amount of credit to top up |
| `API_TOKEN` | Your Papercut API token |

## Usage

```dos
python3.exe topup.py <username>
```
